# 🎮 Tic Tac Toe Multiplayer

A real-time multiplayer Tic Tac Toe game built using **React, Node.js, Express, and Socket.IO**. Players can create or join rooms and play against each other from anywhere in the world.

## 🌐 Live Demo

### Frontend

🔗 https://tic-tac-cj74p09my-prasanna-projects.vercel.app/

### Backend

🔗 https://tic-tac-toe-backend-8auu.onrender.com

---

## 🚀 Features

* 🎮 Real-time multiplayer gameplay
* 🏠 Create and join rooms
* 👤 Player names and automatic X/O assignment
* 🔄 Live synchronization using Socket.IO
* 🏆 Scoreboard
* 🎉 Winner detection with confetti animation
* 🤝 Draw detection
* 🔊 Sound effects
* 🔄 Restart game functionality
* 👥 Online player count
* ❌ Room full protection
* 🔌 Player disconnect handling
* 📱 Responsive design
* ☁️ Deployed using Vercel and Render

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/prasannagollakoti/tic-tac-toe.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
node index.js
```
